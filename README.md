# 🐤 PixelFlappyBird
**유니티로 만든 아기자기한 플래피버드 게임**  
타이밍을 맞춰 화면을 터치하여 파이프 사이를 통과해보세요!💪

## 개발환경
* C#, Unity
* OS: Windows 10
* IDE: Visual Studio 2017
* 개발기간: 2018.11.24
